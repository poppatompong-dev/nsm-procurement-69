## กฎความปลอดภัยเชิงโครงสร้างในการโค้ด (Coding Guards)
* **การจัดการฟอร์แมตตัวเลข:** เนื่องจากอาจพบค่าเป็น `null`, `undefined` หรือชนิดตัวแปรที่เป็นข้อความ (String) ในฟิลด์ราคาและจำนวน ให้เรียกผ่านฟังก์ชันครอบ `formatNumber(val)` เสมอแทนการเรียกใช้ `.toLocaleString()` ตรงๆ เพื่อหลีกเลี่ยงเหตุการณ์ **"หน้าจอขาว" (White Screen Crash)**
* **ตัวกรองกลุ่มงานประชาสัมพันธ์:** ฟิลด์ข้อมูล `division` ใน JSON ใช้ค่าปนกันระหว่างคำเต็มและตัวย่อ (เช่น "ปชส. 3" และ "ประชาสัมพันธ์") ดังนั้นในการกรองกลุ่มงาน "ประชาสัมพันธ์" ใน `filteredItems` ต้องทำกลไกเช็คคู่ทั้งสองคำคู่กันเสมอ
* **การแชร์ข้อมูล:** การอ่านและจัดเก็บรายชื่อคณะกรรมการตรวจรับเริ่มต้นจาก LocalStorage และ URL Hash ต้องมีการตรวจสอบโครงสร้างและอาร์เรย์ (Array Structure Validation) ทุกครั้งก่อนเข้าสู่ `state` ป้องกันข้อมูลที่บีบอัดเกิดความเสียหายจากการคัดลอกลิงก์ไม่ครบถ้วน
