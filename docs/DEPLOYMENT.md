# Deployment & Installation Guide (คู่มือติดตั้งและเผยแพร่)
**ระบบตรวจสอบและรายงานพัสดุคอมพิวเตอร์ - เทศบาลนครนครสวรรค์**

## 1. วิธีติดตั้งเพื่อรันบนบราวเซอร์ของนักพัฒนา (Local Development Setup)

### ข้อกำหนดก่อนติดตั้ง (Prerequisites)
* ติดตั้ง [Node.js](https://nodejs.org/) (แนะนำเวอร์ชัน 18 ขึ้นไป)
* โฟลเดอร์รูปภาพ `รูปภาพ/` ต้องถูกวางไว้ในโฟลเดอร์ `public/รูปภาพ/` ของโปรเจกต์เพื่อให้ระบบอ้างอิงภาพได้สมบูรณ์

### ขั้นตอนติดตั้ง
1. แตกไฟล์โปรเจกต์และเปิดด้วย Terminal
2. ติดตั้งแพ็กเกจที่จำเป็นทั้งหมด:
   ```bash
   npm install
   ```
3. สตาร์ตระบบเครื่องเซิร์ฟเวอร์จำลอง:
   ```bash
   npm run dev
   ```
4. เปิดบราวเซอร์ไปที่ลิงก์ที่แสดงบนหน้าจอ Terminal (เช่น `http://localhost:5173`)

## 2. การสร้างไฟล์สำหรับขึ้นโฮสติ้งจริง (Production Build)
ทำความสะอาดและสร้างโค้ดส่วนหน้าที่มีการคอมไพล์แล้ว:
```bash
npm run build
```
เมื่อรันสำเร็จ โฟลเดอร์ `dist/` จะถูกสร้างขึ้น ซึ่งประกอบด้วยไฟล์ `index.html`, โค้ด Javascript และ Stylesheet ที่ผ่านการย่อขนาด (Minified) พร้อมนำขึ้นโฮสติ้งจริง

## 3. การ Deploy ขึ้น Firebase Hosting (การอัปเดตออนไลน์)
โปรเจกต์นี้ได้รับการผูกไว้กับ Firebase Hosting ภายใต้ชื่อ **`nsm-procurement-69`**

### ขั้นตอนอัปเดตขึ้นออนไลน์:
1. หากยังไม่ติดตั้ง Firebase CLI ให้ติดตั้งผ่าน npm:
   ```bash
   npm install -g firebase-tools
   ```
2. รันคำสั่ง Deploy เพื่ออัปเดตหน้าเว็บบนออนไลน์ (จะใช้ไฟล์ในโฟลเดอร์ `dist/` ขึ้นระบบอัตโนมัติ):
   ```bash
   npx firebase-tools deploy --only hosting --project nsm-procurement-69
   ```
3. เมื่อ Deploy สำเร็จ หน้าเว็บออนไลน์จะพร้อมใช้งานทันทีที่: **[https://nsm-procurement-69.web.app](https://nsm-procurement-69.web.app)**
