# Changelog (ประวัติการเปลี่ยนแปลงและการอัปเดตระบบ)
**ระบบตรวจสอบและรายงานพัสดุคอมพิวเตอร์ - เทศบาลนครนครสวรรค์**

---

## [v1.1.0] - 2026-07-21
### Added (สิ่งที่เพิ่มเข้ามา)
* เพิ่มฟังก์ชันครอบความปลอดภัยในการฟอร์แมตตัวเลข `formatNumber(val)` เพื่อป้องกันข้อผิดพลาดในการโหลดหรือแปลงค่าราคากลางและงบประมาณ ป้องกันปัญหารันไทม์หน้าขาว
* เพิ่มฟังก์ชันพิมพ์รายงาน PDF `handlePrint` เพื่อตอบสนองเหตุการณ์คลิกสั่งพิมพ์รายงานได้อย่างมีประสิทธิภาพ
* เพิ่มการตรวจสอบความปลอดภัยของโครงสร้างข้อมูลรายชื่อคณะกรรมการ (Committee State Validation) ทั้งจาก LocalStorage และ URL Hash บีบอัด เพื่อความถูกต้องในการใช้งานระดับบราวเซอร์

### Changed (สิ่งที่เปลี่ยนแปลง)
* ปรับเปลี่ยนรายชื่อคณะกรรมการตรวจรับเป็นรายชื่อข้าราชการตามบันทึกคำสั่งจัดซื้อจริงของเทศบาลนครนครสวรรค์ ที่ ๘๖๔/๒๕๖๙ (นายณัฏฐวุฒิ จีนมหันต์, นายปฐมพงษ์ หล้ามหศักดิ์, นายประชารักษ์ ประทุมโทน)
* เปลี่ยนวันตรวจรับพัสดุคอมพิวเตอร์เป็นวันที่ **21 กรกฎาคม 2569** (21 ก.ค. 2569) ทั่วทั้งระบบ
* เปลี่ยนแปลงการจับคู่ไฟล์ภาพพัสดุให้ตรงตามความเป็นจริง 100% จากการตรวจสอบเชิงลึก โดยระบุภาพรายการ 1-41, 43, 44-45 และ 46-49 ตรงตามรูปพัสดุในกล่อง
* ปรับปรุงฟิลเตอร์ "กลุ่มงานประชาสัมพันธ์" ในการออกรายงานให้สามารถคำนวณและดึงรายการที่ระบุผู้เบิกเป็น "ปชส." และ "ประชาสัมพันธ์" มารวมแสดงในกลุ่มงานเดียวกันได้อย่างครบถ้วน

### Fixed (สิ่งที่แก้ไข)
* แก้ไขปัญหาหน้าต่างระบบออกรายงานล่มและขึ้น "หน้าขาว" เมื่อกรรมการกดสลับแท็บ เนื่องจากความเสียหายในตรรกะตัวเลขของฐานข้อมูลและ ReferenceError ของปุ่มพิมพ์งาน
* แก้ไขปัญหาลิงก์จับคู่ของปลั๊กม้วนโรลสายไฟ (รายการที่ 42) ที่ภาพหลุดหาย โดยเว้นช่องรูปภาพเป็นค่าว่างเพื่อแยกแยะว่าเป็นพัสดุเดียวที่ยังขาดรูปหลักฐานหน้างาน

## [v1.0.0] - 2026-07-20
* จัดตั้งระบบตรวจสอบพัสดุคอมพิวเตอร์ 49 รายการสำหรับเทศบาลนครนครสวรรค์
* สร้างอินเทอร์เฟซแบบ Responsive มีแดชบอร์ดกราฟสรุปสถิติงบประมาณ และหน้าจับคู่รูปพัสดุแบบคลิกเปลี่ยนไดนามิก
* ติดตั้งระบบแชร์สถานะการตรวจรับ Base64 URL Sharing เพื่อทำงานร่วมกันได้โดยไม่ต้องใช้ระบบคลาวด์ภายนอก
* ติดตั้งระบบพิมพ์เอกสารและส่งออกรายงาน Excel สรุปงบประมาณ
