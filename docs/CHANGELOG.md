# Changelog (ประวัติการเปลี่ยนแปลงและการอัปเดตระบบ)
**ระบบตรวจสอบและรายงานพัสดุคอมพิวเตอร์ - เทศบาลนครนครสวรรค์**

---

## [v1.2.0] - 2026-07-22
### Added (สิ่งที่เพิ่มเข้ามา)
* **Photo Evidence Annex in Official Report**: เพิ่มระบบภาคผนวกรูปภาพหลักฐานพัสดุจริงจากการตรวจรับ 49 รายการในหน้าออกรายงาน สามารถเลือกเปิด/ปิดการแนบรูปภาพได้ตามต้องการ
* **Print PDF with Attached Evidence**: ฟังก์ชันสั่งพิมพ์ PDF รองรับการพิมพ์หนังสือรายงานสรุปทางการ พร้อมลายเซ็นคณะกรรมการ และภาคผนวกรูปภาพพัสดุจริงลงในเอกสาร A4 แผ่นถัดไปได้อย่างสวยงามครบถ้วน
* **1-Click Quick Status Action Buttons**: เพิ่มปุ่มเลือกผลการตรวจรับด่วน 🟢 ผ่าน / 🔴 ไม่ผ่าน / 🟡 รอตรวจ บนการ์ดพัสดุหน้าหลัก กดเปลี่ยนสถานะได้ในคลิกเดียวโดยไม่ต้องเปิดป๊อปอัพ
* **Robust Image Path Helper (`src/utils/imageHelper.js`)**: เพิ่มตัวจัดการการดึง Relative Path ของรูปภาพพัสดุ ป้องกันปัญหาภาพไม่แสดงผลเมื่อนำไปใช้งานบนเว็บโฮสติ้งต่าง ๆ

### Changed (สิ่งที่เปลี่ยนแปลง)
* **Consolidated Navigation (3 Core Tabs)**: ยุบรวมโครงสร้างเมนูหลักจากเดิม 8 เมนูให้เหลือ 3 เมนูหลัก ได้แก่:
  1. `📋 บันทึกตรวจรับพัสดุ` (Inspection & Items)
  2. `📄 ออกรายงาน & พิมพ์ PDF` (Official Report + Photo Annex)
  3. `⚙️ ตั้งค่าโครงการ & กรรมการ` (Project & Committee Settings + Image Mapper + Excel Importer)
* **Typography & Contrast Overhaul**: ขยายขนาดตัวหนังสือทั่วทั้งระบบจากเดิม 9-10px เป็น 14-18px อ่านง่าย สบายตา เหมาะสำหรับเจ้าหน้าที่และคณะกรรมการตรวจรับ
* **Simplified Item Detail Modal**: ปรับปรุงหน้าต่างตรวจรับพัสดุรายชิ้นให้กระชับ เน้นการแสดงรูปภาพพัสดุขนาดใหญ่ สเปกทางการ ปุ่มสถานะตรวจรับ และช่องกรอก Serial Number / หมายเหตุ
* **100% Image Mapping Matrix**: ตรวจทานและแก้ไขการจับคู่รูปภาพพัสดุจริงครบถ้วน 49 รายการ แก้ไขรูปภาพหลุดเลื่อนของรายการที่ 19, 20, 30, 31, 32, 48 และ 49

### Deployment & Synchronization
* **GitHub Repository**: ซิงก์โค้ดรุ่นล่าสุดขึ้นที่ `https://github.com/poppatompong-dev/nsm-procurement-69` (Branch: main)
* **Live Web App**: เผยแพร่เวอร์ชันใช้งานจริงบนคลาวด์ Firebase Hosting ที่ `https://nsm-procurement-69.web.app`

## [v1.1.0] - 2026-07-21
### Added (สิ่งที่เพิ่มเข้ามา)
* เพิ่มฟังก์ชันครอบความปลอดภัยในการฟอร์แมตตัวเลข `formatNumber(val)` เพื่อป้องกันข้อผิดพลาดในการโหลดหรือแปลงค่าราคากลางและงบประมาณ ป้องกันปัญหารันไทม์หน้าขาว
* เพิ่มฟังก์ชันพิมพ์รายงาน PDF `handlePrint` เพื่อตอบสนองเหตุการณ์คลิกสั่งพิมพ์รายงานได้อย่างมีประสิทธิภาพ
* เพิ่มการตรวจสอบความปลอดภัยของโครงสร้างข้อมูลรายชื่อคณะกรรมการ (Committee State Validation) ทั้งจาก LocalStorage และ URL Hash บีบอัด เพื่อความถูกต้องในการใช้งานระดับบราวเซอร์

### Changed (สิ่งที่เปลี่ยนแปลง)
* ปรับเปลี่ยนรายชื่อคณะกรรมการตรวจรับเป็นรายชื่อข้าราชการตามบันทึกคำสั่งจัดซื้อจริงของเทศบาลนครนครสวรรค์ ที่ ๘๖๔/๒๕๖๙ (นายณัฏฐวุฒิ จีนมหันต์, นายปฐมพงษ์ หล้ามหศักดิ์, นายประชารักษ์ ประทุมโทน)
* เปลี่ยนวันตรวจรับพัสดุคอมพิวเตอร์เป็นวันที่ **21 กรกฎาคม 2569** (21 ก.ค. 2569) ทั่วทั้งระบบ
* เปลี่ยนแปลงการจับคู่ไฟล์ภาพพัสดุให้ตรงตามความเป็นจริง 100% จากการตรวจสอบเชิงลึก โดยระบุภาพรายการ 1-41, 43, 44-45 และ 46-49 ตรงตามรูปพัสดุในกล่อง
* ปรับปรุงฟิลเตอร์ "กลุ่มงานประชาสัมพันธ์" ในการออกรายงานให้สามารถคำนวณและดึงรายการที่ระบุผู้เบิกเป็น "ปชส." และ "ประชาสัมพันธ์" มารวมแสดงในกลุ่มงานเดียวกันได้อย่างครบถ้วน

### Fixed (สิ่งที่แก้ไข)
* แก้ไขปัญหาหน้าต่างระบบออกรายงานล่มและขึ้น "หน้าขาว" เมื่อกรรมการกดสลับแท็บ เนื่องจากความเสียหายในตรรกะตัวเลขของฐานข้อมูลและ ReferenceError ของปุ่มพิมพ์งาน
* แก้ไขปัญหาลิงก์จับคู่ของปลั๊กม้วนโรลสายไฟ (รายการที่ 42) ที่ภาพหลุดหาย โดยเว้นช่องรูปภาพเป็นค่าว่างเพื่อแยกแยะว่าเป็นพัสดุเดียวที่ยังขาดรูปหลักฐานหน้างาน
